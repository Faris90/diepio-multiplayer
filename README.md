A Diep.io Clone With Alot Of **features**

Built with **socket.io**
## Installation

1. Go to src
2. Start **installModulles.bat**
3. Once process is done, you can now start **start.bat**

### Instalation (Manual)

1. In Project Folder: **npm install**
2. **cd** to **src**
3. do **node server.js**
